#
#   hipporag   ->  pip install torch, pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu130 , do not mix with the CPU VERSION!!!!
#   graphrag   ->  pip install graphrag lancedb
#
# Forbidden in the base file (still): vllm, transformers, sentence-transformers,
# huggingface-hub, gritlm, boto3, umap-learn. Anything that pulls model weights
# from HuggingFace or requires a GPU does not belong in this base stack.
